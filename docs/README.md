# 프로젝트 문서

이 폴더는 시각장애인 보행 유도 AMR의 설계, 통신, 안전 및 시험 관련 기술 문서를 관리한다.

## 문서 목록

| 문서 | 설명 |
|---|---|
| `system-architecture.md` | 전체 시스템 구성과 데이터 흐름 |
| `protocol.md` | Jetson–STM32 통신 규격 |
| `safety-state-table.md` | 안전 상태와 전환 조건 |
| `hardware-interface.md` | 센서·제어기·배선 인터페이스 |
| `test-plan.md` | 단위·통합·안전 시험 계획 |
| `demo-scenario.md` | 최종 시연 순서와 합격 기준 |
| `meeting-notes/` | 회의 결과와 기술적 결정 기록 |

## 문서 작성 규칙

- 결정된 내용과 검토 중인 내용을 구분한다.
- 거리·시간·속도 등의 값에는 단위를 작성한다.
- 안전 임계값에는 근거와 시험 결과를 기록한다.
- 통신 규격 변경 시 Jetson과 STM32 코드를 함께 수정한다.
- 그림이나 표를 변경하면 관련 설명도 함께 수정한다.
- 중요한 기술 결정에는 날짜와 결정자를 기록한다.
