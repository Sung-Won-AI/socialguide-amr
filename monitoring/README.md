# 상태 관제 및 안내 프로그램

이 폴더는 로봇의 주행 상태, 안전 상태, 센서 정보 및 오류를 표시하는 관제 프로그램을 관리한다.

## 주요 기능

- Jetson 또는 로봇 상태 연결
- 현재 안전 상태 표시
- 좌우 바퀴 속도 표시
- 전방 장애물 거리 표시
- 인식 객체와 TTC 표시
- Cliff·ToF 위험 표시
- 배터리 상태 표시
- 통신·모터·센서 오류 표시
- 최근 정지 원인 표시
- 상태 변화 로그 저장
- 상황별 TTS 안내

## 안전 제한

관제 프로그램은 다음 기능을 직접 수행하지 않는다.

- 모터 직접 제어
- STM32 안전 상태 강제 해제
- E-Stop 우회
- 센서 오류 무시
- 긴급정지 후 자동 재출발

관제 프로그램이 종료되더라도 STM32와 Jetson의 안전 기능은 유지되어야 한다.

## 실행

C# 관제 프로그램을 사용하는 경우:

```bash
dotnet restore
dotnet build
dotnet run
